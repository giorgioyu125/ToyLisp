Detailed Description

Represents a memory arena for fast allocation and bulk deallocation.

This structure uses a flexible array member (FAM) for the buffer, which means the arena and its buffer are allocated as a single contiguous block. This improves cache locality and reduces allocation overhead.

Warning
The arena itself must be heap-allocated (via arena_init()). You cannot declare an Arena on the stack.
When the arena grows via realloc(), the address of the Arena struct itself may change. Always use the returned pointer from allocation functions.
