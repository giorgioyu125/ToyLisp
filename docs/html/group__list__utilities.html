Detailed Description

Functions for operating on Lisp-style lists.

These helpers provide common operations for lists, which are represented as chains of ConsCells terminating in nil.

size_t list_length ( Value list)

Calculates the proper length of a Lisp list.

A proper list is a chain of cons cells ending in nil. This function traverses the list, counting the cells. If the list is improper (does not end in nil), it counts up to the non-cons cdr.

Parameters
listThe list Value to measure.
Returns
The number of cons cells in the list chain.
