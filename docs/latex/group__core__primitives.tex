\doxysubsection{Detailed Description}
A collection of all primitive functions and special forms built into the interpreter. 

This group contains the C implementation of all the language\textquotesingle{}s fundamental operations. Given their basic nature and standard behavior (mirroring that of Lisp/\+Scheme), they are not documented individually.

They are divided into two main categories\+:


\begin{DoxyEnumerate}
\item {\bfseries{Normal Primitive Functions}}\+: Includes operations like {\ttfamily +}, {\ttfamily cons}, {\ttfamily car}, {\ttfamily eq?}, etc. They follow standard evaluation rules\+: all their arguments are evaluated {\itshape before} the function itself is executed.
\item {\bfseries{Special Forms}}\+: Includes constructs like {\ttfamily if}, {\ttfamily define}, {\ttfamily lambda}, {\ttfamily quote}, {\ttfamily and}, {\ttfamily or}. These constructs {\bfseries{do not}} follow the standard evaluation rules and control {\itshape how} and {\itshape if} their operands are evaluated. They are essential for implementing control flow, variable definitions, and fundamental abstractions.
\end{DoxyEnumerate}

All of these functions share the same C signature\+: {\ttfamily \doxylink{structValue}{Value} func(\+Value args, Value env)}.
