Functions for operating on Lisp-\/style lists. 

These helpers provide common operations for lists, which are represented as chains of {\ttfamily \doxylink{structConsCell}{Cons\+Cell}}s terminating in {\ttfamily nil}.

Calculates the proper length of a Lisp list. 

A proper list is a chain of {\ttfamily cons} cells ending in {\ttfamily nil}. This function traverses the list, counting the cells. If the list is improper (does not end in {\ttfamily nil}), it counts up to the non-\/cons cdr.
