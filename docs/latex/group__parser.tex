\doxysubsubsection{Detailed Description}
Functions for parsing Lisp S-\/expressions from a text stream. 

This is a simple recursive descent parser that tokenizes the input and builds {\ttfamily \doxylink{structValue}{Value}} objects representing the abstract syntax tree.
