\doxysubsection{Detailed Description}
A set of factory functions for creating {\ttfamily \doxylink{structValue}{Value}} objects. 

These functions encapsulate the logic for allocating memory (where necessary) and correctly setting the type tag and data for various {\ttfamily \doxylink{structValue}{Value}} types. They are the primary way to create new values within the interpreter.
